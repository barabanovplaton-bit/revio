## ЧТО ЭТО ЗА ПРОЕКТ
Платформа для фрилансеров, где:
- Фрилансер загружает картинки (макет)
- Генерируется ссылка для клиента
- Клиент ставит точечные/общие маячки на картинке
- Фрилансер видит метки в реалтайме
- Система кругов правок (лимиты)

## ТЕХНИЧЕСКИЙ СТАК
- Next.js 16 + React 19 + TypeScript
- Firebase (Firestore, Auth) — ключи уже есть
- Cloudinary (хранение картинок) — ключи уже есть
- Tailwind CSS + Framer Motion
- Деплой: Vercel + GitHub

## ТАРИФЫ
- **Free:** 1 проект, 5 картинок, 1 круг правок
- **Pro 299₽/мес:** безлимит проектов, безлимит картинок, 5 кругов правок

## ПЛАН ДЕЙСТВИЙ (обновлён 16.07.2025)

### ШАГ 1 — Дизайн + настройки + уведомления [ТЕКУЩИЙ]
- Шапка: sticky, тоньше, логотип слева, аватар справа, колокольчик
- Кнопка "Новый проект" — белая, побольше
- Поиск — уже по ширине
- Меню аватара: имя, email, тариф (клик → /pricing), настройки, выйти
- Настройки: тема (свет/тёмная), имя, "О сервисе"
- Колокольчик: красная точка + текст "Клиент отправил правки"

### ШАГ 2 — Страница тарифов
- Отдельная страница `/pricing`
- 2 тарифа: Free / Pro 299₽
- Маркетинговый текст + кнопка "Оплатить"
- Ссылка из настроек и popover аватара

### ШАГ 3 — Создание проекта
- Переработать модалку создания
- После создания → копирование ссылки для клиента

### ШАГ 4 — Уведомления (реалтайм)
- Когда клиент отправил правки → красная точка на колокольчике
- Текст в выпадающем списке
- Клик → переход в проект

### ШАГ 5 — Тест + деплой
- Запустить на Vercel
- Найти 1 тестового фрилансера
- Собрать фидбек

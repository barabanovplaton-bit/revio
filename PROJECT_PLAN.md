# Revio — План проекта

## Что это
SaaS для фрилансеров: загружаешь макеты, отправляешь ссылку клиенту, клиент ставит метки на картинках и пишет комментарии. Фрилансер видит все правки в одном месте.

## Текущий статус: MVP готов, дорабатываем логику

## Жизненный цикл проекта

```
Создание → waiting_for_images → in_progress → exhausted → (архив)
```

### Детали:

1. **Создание** → `waiting_for_images`
   - Фрилансер создаёт проект через визард (название, описание, клиент, число кругов)
   - Картинок нет, ссылка неактивна

2. **Загрузка картинок** → `in_progress`
   - Фрилансер загружает макеты (drag-n-drop / file picker → подтверждение → Cloudinary)
   - Картинки **замораживаются** — после загрузки нельзя добавить/удалить/заменить
   - Ссылка для клиента становится активной

3. **Клиент отправляет правки** → `isLocked = true`, `roundsLeft--`
   - Клиент открывает ссылку `/review/[id]`, ставит метки, жмёт «Отправить»
   - Клиент **блокируется** на этом круге (не может отправить повторно)
   - Фрилансеру приходит уведомление

4. **Фрилансер начинает следующий круг** → `isLocked = false`, `currentRound++`
   - Фрилансер смотрит метки, правит макеты
   - Нажимает «Начать следующий круг» → клиент снова может ставить метки
   - Картинки остаются те же

5. **Круги закончились** → `exhausted`
   - `roundsLeft === 0`
   - Клиент видит «Лимит правок исчерпан»
   - Фрилансер видит все пакеты правок

## Страницы

### `/` — Главная (список проектов)
- Поиск, фильтрация
- Карточки проектов: название, описание/клиент, статус
- Меню: переименовать, закрепить, архив, удалить
- Создание нового проекта

### `/project/[id]` — Детали проекта (freelancer view)
- **Пустое состояние** → big dropzone для загрузки картинок
- **Есть картинки** → грид картинок + секция «Пакеты правок»
- **Клиент отправил** → красный бейдж, кнопка «Начать следующий круг»
- **Все круги исчерпаны** → надпись, ссылка неактивна
- Шапка: назад + название + шестерёнка (настройки)

### `/project/[id]/settings` — Настройки проекта
- Название, описание, клиент, контакт
- **Число кругов** (можно изменить, влияет на roundsTotal)
- Архив / Завершить / Удалить

### `/review/[id]` — Страница клиента
- Проверяет статус проекта
- `waiting_for_images` → «Проект подготовлен, ожидает загрузки макетов»
- `in_progress` + `isLocked` → «Правки отправлены, ожидайте следующего круга»
- `exhausted` → «Лимит правок исчерпан»
- `in_progress` + `!isLocked` → MarkerCanvas (просмотр картинок + метки + отправка)

### `/login` — Авторизация (Google)
### `/notifications` — Уведомления
### `/pricing` — Тарифы
### `/settings` — Глобальные настройки профиля
